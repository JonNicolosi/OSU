To compile chatclient:

gcc -o chatclient chatclient.c

To compile chatserve:

javac chatserve.java
_________________________________

First run chatserve on one flip server with the command: java chatserve <port>

Run chatclient on another flip server with the command: ./chatclient <server address> <port>

_________________________________

After the client gives his/her name, the client will be prompted to send the first message.
