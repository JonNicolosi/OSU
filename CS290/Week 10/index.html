What is an API?


API stands for Application Programming Interface.
API's provide us with functions that we can use to pull data from a website (or any service hosted on the web). We can then use that data to build new applications.

What We Will Learn

  • How to get an API key
  • How to make requests to the API
  • How to parameterize our queries
  • How to loop through an object and extract the information we need

What is Quandl?


Quandl is a website that provides financial and economic data. It features free and premium databases in CSV, JSON, and XML format. In this tutorial we will explore how to retrieve and parse JSON objects by making calls to the Quandl API.
