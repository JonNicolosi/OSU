Signing up at Quandl:


Quandl can be used without an API key. But using Quandl without API will subject you to strict rate limits. You can sign up with an E-mail account, a Google+ account, a LinkedIn account, or a GitHub account.

In account settings, you can navigate to the API key screen from the menu on the left-hand side of the screen. Your current API key is displayed here. You will need to include this key in your API calls.
